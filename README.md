# Breast Cancer Ultrasound Image Classifier

A machine learning project that automates the classification of breast ultrasound images using deep learning techniques, built as an individual project at the University of Surrey.

## Tech Stack

- **Language:** Python
- **Libraries:** PyTorch, NumPy, Matplotlib, scikit-learn
- **Environment:** Jupyter Notebook

## Project Overview

Developed a CNN-based classifier to analyse breast ultrasound images, exploring various techniques to optimise model performance.

## Key Features

- CNN model training and evaluation with AUC metrics
- Hyperparameter tuning (learning rate, epochs, weight decay)
- Data augmentation (random rotations, horizontal flips)
- Optimiser comparison (Adam vs SGD) with automatic best-model selection
- Custom CNN architecture with convolutional and fully-connected layers
- Cross-validation and performance visualisation

## About

Built as a solo university project exploring machine learning classification techniques applied to medical imaging data.
